# Project Overview

This is a personal landing page for "Vintec", a brand focused on digital products and tech education, specifically in Machine Learning and AI. The application is built with React and Vite, and written in TypeScript.

## Building and Running

### Development

To run the development server, use the following command:

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

### Build

To build the project for production, use the following command:

```bash
npm run build
```

The production files will be located in the `dist` directory.

### Preview

To preview the production build locally, use the following command:

```bash
npm run preview
```

## Development Conventions

### Alias

The project uses the `@` alias for the `src` directory.

### Styling

The project uses Tailwind CSS for styling, as inferred from the class names in `App.tsx`.
